   - Create `TimestampMixin` with:
     - `created_at: Mapped[datetime.datetime]` = `mapped_column(DateTime(timezone=True), server_default=func.now(), nullable=False)`
     - `updated_at: Mapped[datetime.datetime]` = `mapped_column(DateTime(timezone=True), server_default=func.now(), onupdate=func.now(), nullable=False)`

2. Create `src/civpulse_geo/models/enums.py`:
   ```python
   import enum

   class LocationType(str, enum.Enum):
       ROOFTOP = "ROOFTOP"
       RANGE_INTERPOLATED = "RANGE_INTERPOLATED"
       GEOMETRIC_CENTER = "GEOMETRIC_CENTER"
   ```

3. Create `src/civpulse_geo/models/address.py` — the `Address` model:
   - `__tablename__ = "addresses"`
   - `id: Mapped[int]` = `mapped_column(primary_key=True)`
   - `original_input: Mapped[str]` = `mapped_column(Text, nullable=False)` — the freeform input as received
   - `normalized_address: Mapped[str]` = `mapped_column(Text, nullable=False)` — USPS-standardized canonical form
   - `address_hash: Mapped[str]` = `mapped_column(String(64), nullable=False, unique=True, index=True)` — SHA-256 hex of normalized_address for fast lookups
   - Parsed components (all nullable String):
     - `street_number: Mapped[str | None]` = `mapped_column(String(20))`
     - `street_name: Mapped[str | None]` = `mapped_column(String(200))`
     - `street_suffix: Mapped[str | None]` = `mapped_column(String(20))`
     - `street_predirection: Mapped[str | None]` = `mapped_column(String(5))`
     - `street_postdirection: Mapped[str | None]` = `mapped_column(String(5))`
     - `unit_type: Mapped[str | None]` = `mapped_column(String(20))`
     - `unit_number: Mapped[str | None]` = `mapped_column(String(20))`
     - `city: Mapped[str | None]` = `mapped_column(String(100))`
     - `state: Mapped[str | None]` = `mapped_column(String(2))`
     - `zip_code: Mapped[str | None]` = `mapped_column(String(5))` — ZIP5 only
   - `base_address_id: Mapped[int | None]` = `mapped_column(ForeignKey("addresses.id"), nullable=True)` — self-referencing FK for unit-to-base relationship (two-tier key)
   - Mixes in `TimestampMixin`
   - Relationships: `geocoding_results` relationship to GeocodingResult, `base_address` self-relationship

4. Create `src/civpulse_geo/models/geocoding.py` with three models:

   **GeocodingResult:**
   - `__tablename__ = "geocoding_results"`
   - `id: Mapped[int]` primary key
   - `address_id: Mapped[int]` = `mapped_column(ForeignKey("addresses.id"), nullable=False, index=True)`
   - `provider_name: Mapped[str]` = `mapped_column(String(50), nullable=False)`
   - `location` = `mapped_column(Geography(geometry_type='POINT', srid=4326), nullable=True)` — use `from geoalchemy2.types import Geography`
   - `latitude: Mapped[float | None]` = `mapped_column(Float, nullable=True)` — denormalized for easy JSON serialization
   - `longitude: Mapped[float | None]` = `mapped_column(Float, nullable=True)` — denormalized
   - `location_type` = `mapped_column(PgEnum(LocationType, name="locationtype", create_type=True), nullable=True)`
   - `confidence: Mapped[float | None]` = `mapped_column(Float, nullable=True)` — 0.0 to 1.0
   - `raw_response: Mapped[dict | None]` = `mapped_column(JSON, nullable=True)` — full provider response
   - Mixes in `TimestampMixin`
   - Add unique constraint on (address_id, provider_name): `__table_args__ = (UniqueConstraint("address_id", "provider_name", name="uq_geocoding_address_provider"),)`

   **OfficialGeocoding:**
   - `__tablename__ = "official_geocoding"`
   - `id: Mapped[int]` primary key
   - `address_id: Mapped[int]` = `mapped_column(ForeignKey("addresses.id"), nullable=False, unique=True)` — one official per address
   - `geocoding_result_id: Mapped[int]` = `mapped_column(ForeignKey("geocoding_results.id"), nullable=False)`
   - Mixes in `TimestampMixin`

   **AdminOverride:**
   - `__tablename__ = "admin_overrides"`
   - `id: Mapped[int]` primary key
   - `address_id: Mapped[int]` = `mapped_column(ForeignKey("addresses.id"), nullable=False, unique=True)` — one override per address
   - `location` = `mapped_column(Geography(geometry_type='POINT', srid=4326), nullable=False)`
   - `latitude: Mapped[float]` = `mapped_column(Float, nullable=False)`
   - `longitude: Mapped[float]` = `mapped_column(Float, nullable=False)`
   - `reason: Mapped[str | None]` = `mapped_column(Text, nullable=True)`
   - Mixes in `TimestampMixin`

5. Create `src/civpulse_geo/models/__init__.py` that imports all models:
   ```python
   from civpulse_geo.models.base import Base, TimestampMixin
   from civpulse_geo.models.enums import LocationType
   from civpulse_geo.models.address import Address
   from civpulse_geo.models.geocoding import GeocodingResult, OfficialGeocoding, AdminOverride

   __all__ = ["Base", "TimestampMixin", "LocationType", "Address", "GeocodingResult", "OfficialGeocoding", "AdminOverride"]
   ```

6. Initialize Alembic:
   - Run `uv run alembic init alembic`
   - Edit `alembic.ini`: set `sqlalchemy.url` to empty string (will be overridden in env.py)
   - Edit `alembic/env.py` to:
     - Import `from civpulse_geo.config import settings`
     - Import `from civpulse_geo.models.base import Base`
     - Import `from civpulse_geo.models import *`  (force model registration)
     - Import `from geoalchemy2 import alembic_helpers`
     - Set `target_metadata = Base.metadata`
     - In `run_migrations_offline()`: use `settings.database_url_sync` and add all three GeoAlchemy2 helpers:
       - `include_object=alembic_helpers.include_object`
       - `process_revision_directives=alembic_helpers.writer`
       - `render_item=alembic_helpers.render_item`
     - In `run_migrations_online()`: use `settings.database_url_sync` via `engine_from_config` override, add the same three helpers
   - Create `alembic/versions/.gitkeep` to ensure the directory exists in git

7. Generate the initial migration:
   - Run `uv run alembic revision --autogenerate -m "initial schema"` (this requires a running DB, so just create the structure — the migration generation will happen when a DB is available, or skip this step and note it for the Docker plan)

NOTE: Do NOT use `declarative_base()` from `sqlalchemy.ext.declarative` — use `DeclarativeBase` from `sqlalchemy.orm`. Do NOT use `Geometry` — use `Geography(geometry_type='POINT', srid=4326)`. Do NOT use `@app.on_event` anywhere.
