**Implementation** (`src/civpulse_geo/normalization.py`):
- Import `hashlib`, `from scourgify import normalize_address_record`, `from scourgify.exceptions import AmbiguousAddressError, AddressNormalizeError`
- `canonical_key(freeform: str) -> tuple[str, str]`:
  1. Call `normalize_address_record(freeform)` inside try/except
  2. On success: extract `address_line_1` and `address_line_2` (city/state/zip) from scourgify output
  3. Parse address_line_1 to separate unit designator from base address — look for unit keywords: APT, STE, UNIT, #, BLDG, FL, RM, DEPT, BSMT, FRNT, REAR, SIDE, LOWR, UPPR, LOT, SPC, TRLR, SLIP, PH, PIER
  4. Build canonical string from: base_address (no unit) + city + state_abbreviation + zip5
  5. Uppercase all components, collapse whitespace
  6. Hash with `hashlib.sha256(normalized.encode()).hexdigest()`
  7. Return `(normalized_string, sha256_hex)`
  8. On scourgify exception: fallback to `_fallback_normalize(freeform)` which does: uppercase, collapse whitespace, strip punctuation except hyphens, hash
- `parse_address_components(freeform: str) -> dict`:
  1. Call `normalize_address_record(freeform)` inside try/except
  2. Parse the result into component dict with keys: `street_number`, `street_name`, `street_suffix`, `street_predirection`, `street_postdirection`, `unit_type`, `unit_number`, `city`, `state`, `zip_code`
  3. Extract zip5 only from postal code (first 5 digits, strip +4)
  4. On exception: return dict with `original_input` key only

IMPORTANT: Use `scourgify.normalize_address_record` NOT `usaddress.parse`. Scourgify wraps usaddress and applies USPS Pub 28 abbreviation tables automatically. The function returns a dict with `address_line_1` and `address_line_2` keys.

`src/civpulse_geo/providers/registry.py`:
```python
from loguru import logger
from civpulse_geo.providers.base import GeocodingProvider, ValidationProvider

def load_providers(provider_classes: dict[str, type]) -> dict[str, GeocodingProvider | ValidationProvider]:
    """Eagerly instantiate all configured providers.

    TypeError is raised here at startup if any provider class omits a required method.
    This satisfies the INFRA-02 requirement that 'a concrete class that omits a required
    method raises an error at load time.'
    """
    registry: dict[str, GeocodingProvider | ValidationProvider] = {}
    for name, cls in provider_classes.items():
        logger.info(f"Loading provider: {name}")
        instance = cls()  # ABC enforcement fires here — TypeError if incomplete
        registry[name] = instance
        logger.info(f"Provider loaded: {name} ({cls.__name__})")
    return registry
```
