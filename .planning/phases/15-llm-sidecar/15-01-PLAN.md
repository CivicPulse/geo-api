       _ZIP_FIRST_DIGIT_STATES = {

    3. **_passes_guardrails(correction, original_state)** function (per D-14):
       - If original_state is not None and correction.state is not None: reject if correction.state.upper() != original_state.upper()
       - If correction.zip and correction.state and len(correction.zip) >= 1: reject if correction.state.upper() not in _ZIP_FIRST_DIGIT_STATES.get(correction.zip[0], set())
       - Return True if all checks pass

    4. **SYSTEM_PROMPT** constant:
       ```python
       SYSTEM_PROMPT = (
           "You are an address parsing and correction assistant for US addresses. "
           "Given a raw address string, extract and correct the address components. "
           "Fix common typos, abbreviations, and formatting issues. "
           "Return the corrected components as JSON.\n\n"
           "Examples:\n"
           "Input: 123 Main St, Macon GA 31201\n"
           '{"street_number": "123", "street_name": "Main", "street_suffix": "St", "city": "Macon", "state": "GA", "zip": "31201"}\n\n'
           "Input: 456 Elm Av Apt 2B, Atlanta Georgia 30301\n"
           '{"street_number": "456", "street_name": "Elm", "street_suffix": "Ave", "city": "Atlanta", "state": "GA", "zip": "30301"}\n\n'
           "Input: 789 Oake Stret, Macn GA\n"
           '{"street_number": "789", "street_name": "Oak", "street_suffix": "St", "city": "Macon", "state": "GA", "zip": null}'
       )
       ```

    5. **LLMAddressCorrector** class:
       - `__init__(self, ollama_url: str, model: str = "qwen2.5:3b")`: stores _ollama_url, _model, _schema = AddressCorrection.model_json_schema()
       - `async def correct_address(self, raw_address: str, http_client: httpx.AsyncClient) -> AddressCorrection | None`:
         - Build payload dict: model=self._model, messages=[system + user], stream=False, format=self._schema, options={"temperature": 0}
         - POST to f"{self._ollama_url}/api/chat" with json=payload and timeout=6.0 (slightly above the 5s asyncio.wait_for to let wait_for handle cancellation)
         - Parse response: resp.json()["message"]["content"]
         - Validate with AddressCorrection.model_validate_json(content)
         - On any exception: log warning with loguru, return None (per D-16 — no retry)

    6. **_ollama_model_available(ollama_url, http_client, model_name="qwen2.5:3b")** async function:
       - GET f"{ollama_url}/api/tags" with timeout=5.0
       - Parse response for models list
       - Return any(m["name"].startswith(model_name) for m in models)
       - On any exception: return False
