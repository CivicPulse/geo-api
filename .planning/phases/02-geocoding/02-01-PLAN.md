    **Step 2: Create GeocodingService**

    Create `src/civpulse_geo/services/geocoding.py` with class `GeocodingService`:

    ```python
    from sqlalchemy.ext.asyncio import AsyncSession
    from sqlalchemy import select
    from sqlalchemy.orm import selectinload
    from sqlalchemy.dialects.postgresql import insert as pg_insert
    import httpx

    from civpulse_geo.normalization import canonical_key, parse_address_components
    from civpulse_geo.models.address import Address
    from civpulse_geo.models.geocoding import GeocodingResult as GeocodingResultORM, OfficialGeocoding
    from civpulse_geo.providers.base import GeocodingProvider
    from civpulse_geo.providers.schemas import GeocodingResult as ProviderResult

    class GeocodingService:
        async def geocode(
            self,
            freeform: str,
            db: AsyncSession,
            providers: dict[str, GeocodingProvider],
            http_client: httpx.AsyncClient,
            force_refresh: bool = False,
        ) -> dict:
    ```

    The `geocode` method implements this exact flow:

    1. **Normalize and hash:** Call `canonical_key(freeform)` -> `(normalized, address_hash)`.
    2. **Find or create Address:** `select(Address).where(Address.address_hash == address_hash)`. If None, create new Address with `original_input=freeform`, `normalized_address=normalized`, `address_hash=address_hash`, plus all fields from `parse_address_components(freeform)` (street_number, street_name, etc.). Call `db.add(address)` then `await db.flush()` to get `address.id`.
    3. **Cache check (unless force_refresh):** `select(GeocodingResultORM).where(GeocodingResultORM.address_id == address.id)`. If results exist, `await db.commit()` and return `{"address_hash": address_hash, "normalized_address": normalized, "results": cached, "cache_hit": True, "official": <official_result_or_None>}`.
    4. **Call providers:** For each provider in `providers.values()`, if it has `geocode` method and is a `GeocodingProvider` instance, call `await provider.geocode(normalized, http_client=http_client)`. The `normalized` string (not `freeform`) must be passed to the provider.
    5. **Upsert results:** For each provider result, use `pg_insert(GeocodingResultORM).values(...)` with `on_conflict_do_update(constraint="uq_geocoding_address_provider", set_={...})`. The EWKT point value is `f"SRID=4326;POINT({result.lng} {result.lat})"` -- CRITICAL: lng comes first in WKT/PostGIS convention. The `location_type` value from Census is the string "RANGE_INTERPOLATED" which matches `LocationType.RANGE_INTERPOLATED.value`. For "NO_MATCH", store None for location_type (it's not in the enum).
    6. **Auto-set official:** Check if `OfficialGeocoding` exists for this address. If not and there are results with a successful match (confidence > 0), create `OfficialGeocoding(address_id=address.id, geocoding_result_id=<first successful result id>)`. Use `pg_insert(OfficialGeocoding).values(...).on_conflict_do_nothing(index_elements=["address_id"])` to avoid races.
    7. **Commit and return:** `await db.commit()`. Return dict with `address_hash`, `normalized_address`, `results` (list of ORM objects), `cache_hit=False`, `official` (the official result or None).

    IMPORTANT: For upsert results, after executing the pg_insert statement, you need the resulting GeocodingResultORM id for the OfficialGeocoding insert. Use `returning(GeocodingResultORM.id)` on the upsert statement, or re-query after flush.

    IMPORTANT: When loading cached results to check for official, use `selectinload` to avoid MissingGreenlet errors: `select(Address).options(selectinload(Address.geocoding_results)).where(...)`.

    IMPORTANT: For "NO_MATCH" results (confidence=0.0), store `location_type=None` in the DB since "NO_MATCH" is not in the LocationType enum. Store `latitude=0.0, longitude=0.0, location=None` for no-match results (no point to store).

    **Step 3: Create geocoding API router**

    Create `src/civpulse_geo/api/geocoding.py`:

    ```python
    from fastapi import APIRouter, Depends, Request
    from sqlalchemy.ext.asyncio import AsyncSession

    from civpulse_geo.database import get_db
    from civpulse_geo.schemas.geocoding import GeocodeRequest, GeocodeResponse, GeocodeProviderResult
    from civpulse_geo.services.geocoding import GeocodingService

    router = APIRouter(prefix="/geocode", tags=["geocoding"])

    @router.post("", response_model=GeocodeResponse)
    async def geocode(
        body: GeocodeRequest,
        request: Request,
        db: AsyncSession = Depends(get_db),
    ):
        service = GeocodingService()
        result = await service.geocode(
            freeform=body.address,
            db=db,
            providers=request.app.state.providers,
            http_client=request.app.state.http_client,
        )
        # Transform ORM results to Pydantic response
        provider_results = [
            GeocodeProviderResult(
                provider_name=r.provider_name,
                latitude=r.latitude,
                longitude=r.longitude,
                location_type=r.location_type.value if r.location_type else None,
                confidence=r.confidence,
            )
            for r in result["results"]
        ]
        official = None
        if result.get("official"):
            o = result["official"]
            official = GeocodeProviderResult(
                provider_name=o.provider_name,
                latitude=o.latitude,
                longitude=o.longitude,
                location_type=o.location_type.value if o.location_type else None,
                confidence=o.confidence,
            )
        return GeocodeResponse(
            address_hash=result["address_hash"],
            normalized_address=result["normalized_address"],
            cache_hit=result["cache_hit"],
            results=provider_results,
            official=official,
        )
    ```
