Output: Two new health routes, lifespan shutdown cleanup, and comprehensive tests for all three behaviors.

    ```python
        if geo_count < 2 or val_count < 2:
            raise HTTPException(
                status_code=503,
                detail={
                    "status": "not_ready",
                    "geocoding_providers": geo_count,
                    "validation_providers": val_count,
                    "reason": "insufficient providers",
                },
            )
        return {
            "status": "ready",
            "geocoding_providers": geo_count,
            "validation_providers": val_count,
        }
    ```

    ```python
    def _install_sigterm_handler() -> None:

        loop = asyncio.get_event_loop()
        def _handle_sigterm():
    ```
