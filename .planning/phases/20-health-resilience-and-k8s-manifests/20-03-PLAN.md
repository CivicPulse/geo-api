<objective>
Create Kustomize overlays for dev and prod with environment-specific patches, ArgoCD Application CRs, and remove obsolete standalone Ollama manifests.

Purpose: Each environment needs its own namespace, ConfigMap values (LLM enabled/disabled, environment name), and ArgoCD Application CR. The old standalone Ollama manifests are replaced by the sidecar in the base Deployment. Secrets are excluded from Kustomize to prevent ArgoCD from overwriting manually-applied credentials.

Output: Complete k8s/overlays/dev/ and k8s/overlays/prod/ directories, ArgoCD CRs, old manifests deleted.
</objective>
