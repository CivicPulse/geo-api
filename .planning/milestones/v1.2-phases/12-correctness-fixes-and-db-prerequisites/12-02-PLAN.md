**A. Tiger county spatial post-filter (FIX-01, per D-01, D-02, D-03):**

Add two new SQL constants at module level in tiger.py:

```python
COUNTY_CONTAINS_SQL = text("""
    SELECT cntyidfp
    FROM tiger.county
    WHERE statefp = :state_fips
      AND ST_Contains(
            the_geom,
            ST_Transform(ST_SetSRID(ST_MakePoint(:lng, :lat), 4326), 4269)
          )
    LIMIT 1
""")

# State abbreviation to FIPS lookup — use tiger.state table
STATE_FIPS_SQL = text("""
    SELECT statefp FROM tiger.state
    WHERE stusps = :state_abbrev
    LIMIT 1
""")
```

Modify `TigerGeocodingProvider.geocode()` method:
1. After the existing `result = await session.execute(GEOCODE_SQL, {"address": address})` and `row = result.first()`, add county containment check BEFORE returning the GeocodingResult.
2. Extract state abbreviation from `row.state` (already returned by GEOCODE_SQL as `(addy).stateabbrev`).
3. Look up state FIPS: `state_result = await session.execute(STATE_FIPS_SQL, {"state_abbrev": row.state})`. If no state found, skip county filter (degrade gracefully).
4. Run containment check: `county_result = await session.execute(COUNTY_CONTAINS_SQL, {"state_fips": state_fips, "lng": row.lng, "lat": row.lat})`.
5. If `county_result.first()` is None — the geocoded point does not fall inside any county in the expected state — return NO_MATCH per D-03.
6. If `county_fips` kwarg is provided (per D-02): verify that the returned `cntyidfp` matches `kwargs.get("county_fips")`. If mismatch, return NO_MATCH.
7. If no `county_fips` kwarg: accept the result as long as it falls in any county in the correct state.

The geocode method should now look like:
```python
async def geocode(self, address: str, **kwargs: Any) -> GeocodingResult:
    try:
        async with self._session_factory() as session:
            result = await session.execute(GEOCODE_SQL, {"address": address})
            row = result.first()

            if row is None:
                return GeocodingResult(...)  # existing NO_MATCH

            # FIX-01: County spatial post-filter (D-01, D-02, D-03)
            if row.state:
                state_result = await session.execute(
                    STATE_FIPS_SQL, {"state_abbrev": row.state}
                )
                state_row = state_result.first()
                if state_row:
                    county_result = await session.execute(
                        COUNTY_CONTAINS_SQL,
                        {"state_fips": state_row.statefp, "lng": row.lng, "lat": row.lat},
                    )
                    county_row = county_result.first()
                    if county_row is None:
                        # D-03: geocode falls outside all counties in expected state
                        return GeocodingResult(
                            lat=0.0, lng=0.0, location_type="NO_MATCH",
                            confidence=0.0, raw_response={}, provider_name=self.provider_name,
                        )
                    # D-02: if caller specified county_fips, verify match
                    expected_county = kwargs.get("county_fips")
                    if expected_county and county_row.cntyidfp != expected_county:
                        return GeocodingResult(
                            lat=0.0, lng=0.0, location_type="NO_MATCH",
                            confidence=0.0, raw_response={}, provider_name=self.provider_name,
                        )

            # ... existing confidence calculation and return
    except SQLAlchemyError as e:
        raise ProviderError(...)
```
