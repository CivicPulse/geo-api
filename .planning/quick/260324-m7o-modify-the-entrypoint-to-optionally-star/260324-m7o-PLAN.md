<task type="auto">
  <name>Task 2: Add conditional debugpy startup to entrypoint and update Compose config</name>
  <files>scripts/docker-entrypoint.sh, docker-compose.yml, .vscode/launch.json</files>
  <action>
1. Edit scripts/docker-entrypoint.sh: Replace the final line (`exec uvicorn civpulse_geo.main:app --host 0.0.0.0 --port 8000`) with a conditional block:

```bash
if [ "${DEBUG:-0}" = "1" ]; then
  echo "Starting API with debugpy on port 5680 (waiting for client)..."
  exec python -m debugpy --listen 0.0.0.0:5680 --wait-for-client -m uvicorn civpulse_geo.main:app --host 0.0.0.0 --port 8000 --reload
else
  exec uvicorn civpulse_geo.main:app --host 0.0.0.0 --port 8000
fi
```

2. Edit docker-compose.yml api service:
   - Add `DEBUG: "1"` to the environment block
   - Add `"5680:5680"` to the ports list
   - Add `./src:/app/src` to the volumes list (for live reload and breakpoint mapping)

3. Create .vscode/launch.json:
```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Attach to Docker (debugpy)",
      "type": "debugpy",
      "request": "attach",
      "connect": {
        "host": "localhost",
        "port": 5680
      },
      "pathMappings": [
        {
          "localRoot": "${workspaceFolder}/src",
          "remoteRoot": "/app/src"
        }
      ],
      "justMyCode": true
    }
  ]
}
```
